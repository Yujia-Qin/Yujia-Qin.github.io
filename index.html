Projects

Here are some research and programming project I have done.

Study length distribution of PCFG and graph size of HRG

A research project on Natural Language Processing with professor David Chiang. We studied the length distribution of sentences generated by PCFG. We developped EM algorithms to train PCFG that model sentence length, and tested the parsing accuracy of the improved grammar. We are now using the technique to improve the HRG that generate social network graphs.

Sarcasm Prediction

Final project for the NLP class. We are trying to predict sarcastic tweets. Our main approach is that sarcastic tweets often contain contrasting emotions.

Predict when people "zone out" while they watch a movie

A research project with professor Sidney DMellow. We used machine learning to study peopleâ€™s mind wandering while they are watching a movie. Mind wandering means people switch from task-related work to non-task related work, so detecting mind wandering is very important for improving task performance. I developed the whole C# application to process the data and produce features and instances; I also used java and machine learning algorithms from Weka to build and evaluate machine learning methods including naive bayes, logistic regression, random forest etc.

Mathfeud Website

A Django website that I developed with another classmate at Notre Dame. It is for a psychology research project to help middle school students learn math. We developed sections for math tutorial videos, discussion forums, question and answers etc.

CMinor Compiler

A compiler witten in C for a toy language CMinor. Implemented scanner, parser, typechecker, and finally code generator. It is for the Compilers class project.
